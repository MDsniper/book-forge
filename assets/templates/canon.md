# Canon (fiction — hard facts DB)

> For nonfiction, use `evidence.md` instead. Every fact here is what's TRUE in the world. A chapter that contradicts canon caps `canon_compliance` at 6.

**Target: 400+ entries before exiting foundation.** Canon grows during drafting — every chapter establishes facts that get logged so future chapters stay consistent.

## Format

| ID | Category | Fact | Source | Notes |
|----|----------|------|--------|-------|
| C001 | geography | The Halberd Stair has 312 steps. | ch_03 | counts vary in-world |
| C002 | timeline | Mira's brother vanished in the Spring Thaw of year 7. | ch_05 | |
| C003 | magic | Salt disrupts the binding spell. | world.md | cost: burns the user |
| C004 | character | Kael is left-handed. | ch_02 | |

## Categories
- **geography** — places, distances, terrain.
- **timeline** — when things happened, in order.
- **magic** — rules, costs, limitations (cross-ref `world.md`).
- **character** — physical facts, abilities, history, relationships.
- **event** — in-story events that cannot be undone (deaths, betrayals, treaties).
- **object** — named objects and their properties.
- **culture** — customs, laws, taboos, rituals.

## Source
- The file/scene that established the fact (`ch_NN`, `world.md`, `characters.md`).
- If retconned, mark the new source and add a note explaining the change.

## Rules
- Every entry must be **falsifiable** (you could write a sentence that violates it).
- If drafting reveals a new fact, **append it here** with the source — don't leave it implicit.
- If a chapter contradicts canon, fix the chapter (almost always right). Rarely, update canon if the chapter revealed a better choice and re-check downstream.
